Move Subtly

Move Subtly is a jQuery plugin that takes any image and turns it into a subtly moving/scaling image.

View the Project on GitHub drewbrolik/Move-Subtly

  • Download TAR Ball

Default Settings

$("#image1").moveSubtly();

Custom Settings

$("#image2").moveSubtly({moveTime:1000,stillTime:6000,lockSide:"bottom"});
